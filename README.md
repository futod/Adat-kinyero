# Adat-kinyero
A munkahelyem miatt készítettem egy adat kinyerőt, amivel sokkal könnyebben és gyorsabban tudunk szervíz címkékett nyomtatni, Trello és M360 integráció az OVIP vállalat irányítási rendszerhez, mivel alapvetően ezeket nem támogatja, azonban szükségünk van egy raktárkezelőre lásd (OVIP), így nem hagyhatjuk el. Az M360 a készülékek tesztelésére szolgál és így látható az OVIP termék adatlapon, hogy egy adott készülék volt-e már korábban tesztelve, avagy sem. A trelloba minden egyéb hozzászólást és státuszt módosítunk, hogy átláthatóbb legyen az egész cég működése. Szerettem volna, ha minden elérhető egy felületen, ezért készítettem ezt a böngészőbe telepíthető bővítménynt.
